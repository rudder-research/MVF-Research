# code/ — VCF Core Engines

This folder contains all systems that process, transform, and score macro + market inputs.

```
code/
│
├── math/               # Core mathematical foundation
├── regime_engine/      # Phase I
├── sector_engine/      # Phase II
├── unified_engine/     # Phase III
├── wavelit_engine/     # Phase IV
├── shared/             # Shared utilities
└── data_normalized/    # Normalized data (intermediate)
```

Each engine is fully modular and can be used independently or combined.
