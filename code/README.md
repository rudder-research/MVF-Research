# VCF Code

All VCF engines and mathematics live here.

## Structure
- `math/` - Core VCF mathematics (Hamiltonian, KAM, geometry)
- `regime_engine/` - Phase I: Market regime detection
- `sector_engine/` - Phase II: Sector rotation analysis
- `unified_engine/` - Phase III: Unified multi-scale framework
- `wavelit_engine/` - Phase IV: Wavelet decomposition
- `shared/` - Shared utilities (I/O, plotting, validation)
- `data_normalized/` - Normalized input data for engines
