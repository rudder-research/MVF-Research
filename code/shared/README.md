# shared/ — Utilities Used Across All Engines

## Includes:

- `io_utils.py` — load/save, file management
- `plotting.py` — shared plotting utilities
- `data_loader.py` — unified loader for raw/normalized data
- `build_macro_panel.py` — aligned macro panel builder
- `geometry_engine.py` — shared geometric toolkit
- `normalize_metrics.py` — normalization utilities

These files provide common I/O, plotting, loaders, and shared transforms.
